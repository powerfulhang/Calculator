# Calculator v2.0.0

**Release Date:** 2026-05-03

## New Features

- **Custom application icon** — calculator icon embedded in window title bar and .exe file
- **Unit test suite** — 77 tests covering the entire expression evaluation engine
- **Project configuration** — `pyproject.toml` with metadata, pytest, ruff, and mypy settings

## Improvements

- **Error messages localized to Chinese** — all exception messages now match the Chinese UI
- **Help content externalized** — documentation moved to dedicated `help_content.py` module
- **Programmer mode operation chaining** — supports `5+3=+2=` workflow like scientific mode
- **UI layout fixes** — eliminated white gaps when switching between calculator modes
- **Author field unified** across all source files

## Technical

- PyInstaller spec updated to bundle icon and new modules
- `generate_icon.py` utility for regenerating the icon
- `.gitignore` and `.gitattributes` properly configured
